Rule MCP Server

Connect Claude and other AI clients to Rule — the marketing automation platform for email & SMS.

1 Get your Rule API key

StepDetails
Log in app.rule.io
Navigate Settings → Developer → New API key
Copy Save the key securely — you will need it below
⚠

Never share your API key or commit it to version control.

2 Configure your MCP client

The server is already running at https://rule-mcp-server-production.up.railway.app/mcp — no installation needed. Select your client below:

Claude Desktop

macOS / Windows app

Claude Code

CLI / VS Code extension

Cursor

AI-native code editor

Open (or create) your Claude Desktop config file:

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

Add the following:

{
  "mcpServers": {
    "rule": {
      "type": "streamable-http",
      "url": "https://rule-mcp-server-production.up.railway.app/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_RULE_API_KEY"
      }
    }
  }
}

Replace YOUR_RULE_API_KEY with your key, then restart Claude Desktop. The Rule tools will appear in the tool picker.

Run this command (replace YOUR_RULE_API_KEY with your key):

claude mcp add rule \
  --transport streamable-http \
  --header "Authorization: Bearer YOUR_RULE_API_KEY" \
  https://rule-mcp-server-production.up.railway.app/mcp

The Rule tools will be available immediately in your next Claude Code session.

Open Cursor Settings → MCP and add a new server entry:

{
  "mcpServers": {
    "rule": {
      "type": "streamable-http",
      "url": "https://rule-mcp-server-production.up.railway.app/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_RULE_API_KEY"
      }
    }
  }
}

Replace YOUR_RULE_API_KEY with your key and save.

3 Verify the connection

Inside your AI client, try asking:

List my Rule subscribers

If the tools are connected you will get a response from your Rule account. You can also check the server status at:

https://rule-mcp-server-production.up.railway.app/health

Available tools

CategoryTools
Subscribers Create, get, update, delete subscribers and their custom fields
Tags Add, remove, and manage subscriber tags segments
Campaigns List, create, and retrieve campaign statistics
Transactions Send transactional emails and SMS messages
Templates List and retrieve email templates
Segments List audience segments

Troubleshooting

ProblemSolution
401 Unauthorized Check that your Authorization: Bearer header contains a valid, active Rule API key
Tools not appearing Restart your MCP client after updating the config file
Connection refused Verify the server is up at rule-mcp-server-production.up.railway.app/health
Wrong transport error Ensure your client uses streamable-http (not stdio) as the transport type
